# BudgetWise
A simple and intuitive personal finance manager that helps users track expenses, manage budgets, and stay financially organized.

# Overview
BudgetWise is a lightweight personal finance management application designed to help users take control of their daily spending. It simplifies the process of tracking expenses, organizing transactions, and monitoring budgets through an intuitive and visually clean interface. The goal is to make financial awareness effortless — whether you're managing pocket money, monthly expenses, or saving toward a goal.
With BudgetWise, users can quickly log transactions, categorize spending, and view summary insights that highlight where their money goes. The app focuses on ease of use and clarity, ensuring that anyone — even without financial knowledge — can maintain their budget confidently.

# Features
Add, Edit, Delete Transactions
Categorize expenses (Food, Travel, Shopping, Bills, etc.)
Monthly budget tracking
Visual analytics (charts for spending distribution)
Simple & responsive UI
Local or cloud-based storage (depending on your implementation)

# Installation & Setup
Clone the repository:
git clone https://github.com/YOUR_USERNAME/BudgetWise.git
cd BudgetWise
Install dependencies:
npm install
Start development server:
npm start
Build for production:
npm run build

# BudgetWise application is hosted on a Google Cloud Compute Engine VM instance, providing secure, scalable, and reliable deployment infrastructure.
